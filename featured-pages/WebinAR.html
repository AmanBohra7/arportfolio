Realtime webpage generator from AR

An android application that provides you a web panel in Augmented Reality(AR) through which you can design a web page in that AR experience only using some Bootstrap component and colors and submit it, then your entire design will be converted into an HTML code using React framework.

The main features the application are :
- Create a unique design using the application and generate your HTML page.
- Provide a full interactive AR experience.
- Realtime HTML component using Bootstrap
- Since using Bootstrap components, you can easily generate responsive page.
